# MPB Website – Company & Business Unit

Repository ini adalah website resmi untuk:
- **PT Multiusaha Prioritas Bersama (Company)**
- **Sekolah Mentor Indonesia (Business Unit)**

## Struktur Folder Utama

- `company/` → Landing page & informasi Company (MPB)
- `smi/` → Website Sekolah Mentor Indonesia
- `public/` → Asset statis (images, icons, dll)
- `src/` → Source code utama

## Catatan Penting

- Project ini menggunakan satu domain dengan pemisahan halaman.
- SEO dioptimalkan untuk Company & Business Unit.
- Mobile responsive & production-ready.
